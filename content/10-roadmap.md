# Program Roadmap

## Roadmap Overview

The 5-year roadmap is organized by fiscal year and quarter, spanning program initiation through full operational capability. The roadmap is a living artifact, updated every PI cycle to reflect actual progress and evolving priorities.

---

## Year 1 (FY2025): Foundation

| Quarter | Key Deliverables |
|---|---|
| Q1 | Program charter signed; PMO established; governance structures activated |
| Q1 | Agency onboarding initiated; agency CIOs briefed; MoUs signed (×12) |
| Q1 | ART teams formed and SAFe 6.0 training completed |
| Q2 | Current State Assessment Report delivered |
| Q2 | Target Architecture Blueprint approved by PCB |
| Q2 | DevSecOps platform deployed and operational |
| Q2 | Cloud landing zones deployed for pilot agencies (3 of 12) |
| Q3 | PI-1 complete: Cloud landing zones for all agencies |
| Q3 | Application Portfolio Assessment (300+ apps) complete |
| Q3 | Zero Trust identity foundation deployed (Enterprise IAM + MFA) |
| Q4 | PI-2 complete: First Tier-3 workload migrations (30 systems) |
| Q4 | Data governance framework established; Data Council inaugural meeting |

**Year 1 Milestone:** Program fully operational, infrastructure foundation in place
